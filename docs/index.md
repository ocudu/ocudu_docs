# OCUDU Project Documentation

![ocudu_logo](https://srs.io/wp-content/uploads/ocudu_color.png)

OCUDU is a permissivly-licensed, open-source 5G (and beyond) CU/DU project desgined for
commercial deployment and broad industry adoption, as well as advanced research and development.
OCUDU is a complete RAN solution compliant with 3GPP and O-RAN Alliance specifications and
includes the full L1/2/3 stack with minimal external dependencies.
OCUDU is govorned under the [Linux Foundation](https://ocudu.org/).
