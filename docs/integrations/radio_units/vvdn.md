# VVDN

## Overview

This guide provides further details on using OCUDU with O-RUs from [VVDN](https://www.vvdntech.com/).

---

## Configuration

### CU/DU

You can download a sample gNB configuration file that is compatible with the indoor VVDN RU [here](assets/gnb_ru_vvdn_tdd_n78_100mhz.yml).

This configuration file will allow you to create a 100MHz SISO TDD cell in band n78.
