# Netgear

Netgear produces a range of switches with PTP support suitable for use as timing-aware O-RAN fronthaul switches. In general, any Netgear device with announced PTP support should be compatible — see the [Netgear website](https://www.netgear.com) for the full list of supported devices.

The following devices have been tested in-house with OCUDU:

- **Netgear M4300-8X8F**
