# COTS UEs

This section of our knowledge base aims to list COTS UEs known to be working with OCUDU including tested bands and configurations.
It's not an exhaustive list and many more UEs will work out of the box even though they are not listed in this document.

However, if you have done extensive testing with a particular phone and want to contribute your findings please don't hesitate to
submit a merge request.
