# User Manual

Use this section to install, configure, and operate OCUDU. The guides here assume you have a specific deployment goal; if you are new to OCUDU and want a structured walkthrough from scratch, start with the [Tutorials](../tutorials/index.md) instead.

:::note
OCUDU requires a Linux host with RT Kernal. For example: Ubuntu, Fedora, RedHat, SUSE, Arch. 
:::

**For a first deployment**, work through the sections in this order:

1. **Installation**: install dependencies, build OCUDU from source, and set up your RF hardware or drivers
2. **Running**: start the gNB and verify connectivity to a 5G core network
3. **Configuration Reference**: tune parameters to match your hardware and network requirements

The remaining sections address specific operational topics: use **Outputs** and **Grafana** to monitor a running gNB, and **Troubleshooting** when something goes wrong.
